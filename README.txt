This program extracts rotten tomatoes and imdb info to portray various things as the user commands. It can 1) show the imdb and rotten tomatoes score for a specific movie the user inputs 2) print out the top 30 movies in theaters with the name and both imdb and rotten tomatoes score 3) user sets which range (ex: 2-5 would be the top 2-5 movies in theaters) and portray it in a bar graph based on score
